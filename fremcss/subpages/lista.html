Lista uporządkowana

Do stworzenia listy uporządkowanej należy użyć znacznika ol. Wewnątrz tego znacznika do stworzenia elementów listy należy używać znacznika li.

Kod Wynik 
<ol>
    <li>Element listy</li>
    <li>Element listy</li>
    <li>Element listy</li>
</ol>

Lista nieuporządkowana

Do stworzenia listy nieuporządkowanej należy użyć znacznika ul. Wewnątrz tego znacznika do stworzenia elementów listy należy używać znacznika li.

Kod Wynik 
<ul>
    <li>Element listy</li>
    <li>Element listy</li>
    <li>Element listy</li>
</ul>
  • Element listy
  • Element listy
  • Element listy

Połączenie list ze sobą

Jeżeli chcemy umieścić jedną listę w drugiej należy zagnieździć jedną listę w środku drugiej wewnątrz elementu li.

Kod Wynik 
<ul>
    <li>
        <ol<
            <li>Element listy zagnieżdżonej</li>
        </ol<
    </li>
    <li>Element listy głównej</li>
</ul>
